CS 8 Spring 2019 Test 1 - Tower of Hanoi

Team Members: Zijian Zhao, Wei Jeng

Comment:

The Tower of Hanoi assignment is completed by Mr. Zhao and Ms. Jeng. Each person made contributions to the skeletal (data structure) parts of the project, i.e. the priority queue, stack, board, stackboard, disks, and nodes, although it should be noted that Mr. Zhao designed and built the UI and Ms. Jeng does not wish to claim the extra credit.

The game optimizes for the minimum game moves by using the algorithm that relies the even and odd number of disks in the game and also the even or odd disk number on the peg. The UI part is built with with imported functions from the SFML library.
